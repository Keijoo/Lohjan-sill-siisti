# Lohjan sillä siisti 🧹

A business landing page built for a cleaning service ("Lohjan sillä siisti Tmi") operating in Lohja and Western Uusimaa, Finland.

## About

A clean, single-page marketing site presenting the company's cleaning services to local customers, built with plain HTML, CSS, and JavaScript.

## Features

- Responsive, single-page layout
- Service overview
- Branding and iconography
- Contact section

## Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript

## Running Locally

```bash
git clone https://github.com/Keijoo/Lohjan-sill-siisti.git
cd Lohjan-sill-siisti
# open index.html in your browser
```
